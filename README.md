
# Framework

- backend: SpringBoot + JPA + mybatis + mysql
- frontend: webpack + vuejs + element ui

# How to run 

## front 
```
cd front
npm install
npm run dev   # develop mod
npm run build # build the project
```

## backend
```
cd backend
mvn eclipse:eclipse
mvn spring-boot:run
```
